# mcp-korean-spell

`mcp-korean-spell` is a MCP(Model Context Protocol) server for korean spell checking.

## Tools

- `fix_korean_spell`


## Configuration

### npx

```json
{
  "mcpServers": {
    "korean-spell-checker": {
      "command": "npx",
      "args": [
        "-y",
        "@winterjung/mcp-korean-spell"
      ]
    }
  }
}
```
